## Game of Life

[Click here to view](https://dbuzzin.github.io/game-of-life/)

### Current Features

+ Start and stop game from running.
+ Click and drag to add/remove living cells.
  + Button to switch between drawing and erasing.
+ Clear world button.
+ Basic zoom out feature.
